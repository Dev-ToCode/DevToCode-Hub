# DevToCode Hub

DevToCode Hub é um agrupador de links moderno desenvolvido para centralizar projetos, redes sociais e principais canais da marca DevToCode em uma única interface.

O projeto foi construído com foco em uma experiência visual clean, tecnológica e responsiva, utilizando uma identidade inspirada em produtos digitais e startups modernas.

Além de funcionar como uma central de acesso rápido, o DevToCode Hub também fortalece o branding da marca, reunindo portfólio, GitHub, LinkedIn e contatos em uma estrutura organizada, intuitiva e visualmente profissional.

## 🚀 Tecnologias

- HTML5
- CSS3
- JavaScript
